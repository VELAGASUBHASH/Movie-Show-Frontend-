# 🎬 Movie Show App

Live Preview Of Project : https://movieshow-one.vercel.app/

A modern React Movie App built using **TMDB API**.  
Users can browse popular movies, search movies, and add/remove favorites.

---

## ✨ Features

- 🔥 Fetch Popular Movies (TMDB API)
- 🔎 Search Movies
- ❤️ Add / Remove Favorites
- 💾 Favorites stored using Context API
- 🎨 Modern Responsive UI
- 🌙 Clean Component-Based Architecture
- 📱 Fully Responsive (Mobile / Tablet / Desktop)

---

## 🛠 Tech Stack

- React (Vite)
- React Router
- Context API
- TMDB API
- CSS (Grid + Flexbox)
- React Hot Toast
